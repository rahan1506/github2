# Paw & Purr — Pet Boutique (Demo Website)

Open `index.html` in a browser to view the demo site.

Files added:
- index.html — main page
- css/styles.css — styles and animations
- js/app.js — small interactive behaviors
- assets/*.svg — inline SVG illustrations and logo

This is a static demo. To run locally:

1. Open a terminal in this folder.
2. Run a static server, e.g. using Python:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

Enjoy!
